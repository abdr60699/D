# Stellar - Modern User-Friendly Web App

A modern, responsive, and accessible web application built with HTML, CSS, and JavaScript. Stellar delivers an exceptional user experience with beautiful design and powerful functionality.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and intuitive interface with smooth animations
- **Accessible**: Built following accessibility best practices (WCAG guidelines)
- **Fast Performance**: Optimized loading times and smooth interactions
- **Interactive Elements**: Form validation, mobile navigation, smooth scrolling
- **Cross-browser Compatible**: Works seamlessly across all modern browsers

## File Structure

- `index.html` - Main HTML structure with semantic elements
- `styles.css` - Responsive CSS with mobile-first approach
- `script.js` - Interactive JavaScript functionality

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter font family)

## Key Features Implemented

### User Experience
- Sticky navigation header with smooth scrolling
- Mobile-responsive hamburger menu
- Interactive hero section with call-to-action buttons
- Animated feature cards on scroll
- Contact form with real-time validation
- Success modal for form submission
- Active navigation link highlighting

### Accessibility
- Semantic HTML elements (header, nav, section, footer)
- ARIA labels for interactive elements
- Keyboard navigation support
- Focus styles for better keyboard navigation
- Alt text for images and icons
- Proper heading hierarchy

### Performance
- Debounced scroll events for better performance
- CSS animations using transform and opacity
- Optimized intersection observer for animations
- Minimal external dependencies

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required
3. Works offline - all resources are self-contained

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Customization

You can easily customize the website by modifying:

- **Colors**: Edit CSS variables in `styles.css` (`:root` section)
- **Content**: Update text and sections in `index.html`
- **Functionality**: Modify interactions in `script.js`
